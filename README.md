# Python GUI Mini Projects (Tkinter)

A collection of **three Python GUI applications** built using **Tkinter**. These projects demonstrate GUI design, user input handling, and API integration with a clean and modern interface.  

---

## **Projects Included**

### 1. BMI Calculator (`bmi.py`)
**Description:**  
A simple GUI application to calculate **Body Mass Index (BMI)** based on user input for height and weight.  

**Features:**  
- Calculates BMI using the formula: `BMI = weight / (height^2)`  
- Displays BMI category: Underweight, Normal, Overweight, Obese  
- User-friendly input validation  

**Usage:**  
```bash
python bmi.py
2. Password Generator with Strength Indicator (passwordgen.py)

3. Weather App (weather.py)
Description:
Fetches and displays the current weather for a given city using the Open-Meteo API.

Features:

Enter a city name to get current weather

Displays temperature, wind speed, and weather code

Handles invalid input and API errors gracefully

Clean, card-style GUI
